# Milk Dairy Management System (GDMS v10)

## Description
A web-based Milk Dairy Management System built with PHP and MySQL 
that manages farmers, payments, and report generation.

## Features
- Farmer dashboard
- Payment processing (M-Pesa integration)
- Report generation
- Admin panel

## Technologies Used
- PHP
- MySQL
- HTML/CSS
- M-Pesa API (B2C/STK Push)

## Database Setup
1. Import `gdms_v2/assets/milkdiary_db.sql` into your MySQL server
2. Update database credentials in the config file

## How to Run
1. Clone the repository
2. Set up a local server (XAMPP/WAMP)
3. Import the database
4. Open in browser via localhost
